## Installation

- Installed pnpm (tested on 10.14.0 version)
- Build csv files, see [instrutcion](../../README.md)
- Copy files to public

```bash
  mkdir -p ./public/ && cp ../../build/database.csv "$_"
```
- Install npm deps

```bash
  pnpm install
```

## Running local dev

```bash
  pnpm dev
```

## Build

```bash
  pnpm build
```
